THE DEFUSER

The defuser is a game of trust and strategy.

RULES

All players are in a room with a bomb. Multiple wires coming from the bomb, some are GOOD, some are BAD.

Once all the good wires are cut, the bomb is defused. Once all the bad wires are cut or 5 minutes elapses, the bomb explodes!

One player is a DEFUSER, the others are INFORMANTs. Only the defuser can cut wires. Only informants will get information about which wires are good or bad.

Each player (including the defuser) is either GOOD or EVIL. Good players win if the bomb is defused. Evil players win if the bomb explodes.

Informants can vote to kill the defuser. When an evil defuser is killed, good players win. When a good defuser is killed, evil players win.

COMMANDS

Text "new" to create a new game.

Text "join X" to join game #X.

Text "start" to start the game

Text "cut X" to cut wire #X.

Text "kill" vote to kill the defuser.

Text "i am YOURNAME" to change your name.
